## 🔥 What You'll Build

Ever wondered who's the ultimate GitHub contributor? **Mona Mayhem** is a retro arcade-themed battle arena where contribution graphs duke it out in pixelated glory. 

This isn't just a demo — it's a **hands-on workshop** where you'll build this entire app using **GitHub Copilot** as your AI pair programmer. Go from zero to deployed in a single session, learning production-ready AI-assisted development workflows along the way.

### ✨ Features You'll Ship

- 🎯 **Real GitHub Data** — Fetch and visualize actual contribution graphs
- 🎨 **Retro Arcade Theme** — Pixel-perfect 8-bit aesthetics with Press Start 2P font
- ⚡ **Blazing Fast** — Built with Astro v5 for optimal performance
- 🤖 **AI-Powered Development** — Learn to wield GitHub Copilot like a pro
- 🚀 **Production Ready** — Node.js server with proper API routes and error handling

```bash

git clone https://github.com/YOUR-USERNAME/my-mona-mayhem.git
cd my-mona-mayhem

# Install dependencies
npm install
```

### Step 3: Launch the Workshop

**VS Code Users:**
```bash
code .  # Open in VS Code and start the workshop guide
```

**CLI Users:**

- **Node.js 22+** (if installing via npm)

**Install Options:**
```bash
# Via npm
npm install -g @github/copilot

# Via Homebrew (macOS)
brew install github/gh/copilot

# Via WinGet (Windows)
winget install GitHub.Copilot
```

</details>
